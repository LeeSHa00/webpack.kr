생성된 각각의 청크 맨 위에 배너를 추가하세요.

```javascript
const webpack = require('webpack');

new webpack.BannerPlugin(banner);
// or
new webpack.BannerPlugin(options);
```

## Options

```ts
{
  banner: string | function, // 배너를 문자열 또는 함수로 사용하면 주석으로 래핑됩니다.
  raw: boolean, // true이면 배너가 주석에 래핑되지 않습니다.
  entryOnly: boolean, // true인 경우 배너는 항목 청크에만 추가됩니다.
  test: string | RegExp | [string, RegExp], // 테스트 어써션(표명)을 통과하는 모든 모듈을 포함합니다.
  include: string | RegExp | [string, RegExp], // 이러한 조건 중 하나와 일치하는 모든 모듈을 포함합니다.
  exclude: string | RegExp | [string, RegExp], // 이러한 조건과 일치하는 모든 모듈을 제외합니다.
}
```

## Usage

```javascript
import webpack from 'webpack';

// string
new webpack.BannerPlugin({
  banner: 'hello world',
});

// function
new webpack.BannerPlugin({
  banner: (yourVariable) => {
    return `yourVariable: ${yourVariable}`;
  },
});
```

## Placeholders

webpack 2.5.0부터 자리 표시자는 `banner` 문자열에서 평가됩니다.:

```javascript
import webpack from 'webpack';

new webpack.BannerPlugin({
  banner:
    'fullhash:[fullhash], chunkhash:[chunkhash], name:[name], filebase:[filebase], query:[query], file:[file]',
});
```
